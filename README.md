# Symptom Checker Bot

## Problem
People often ignore minor symptoms or panic unnecessarily. There is a need for a simple tool that provides basic health advice.

## Solution
This project is a symptom checker chatbot that takes user symptoms as input and provides basic medical advice.

## Features
- Detects multiple symptoms
- Provides advice
- Handles emergency symptoms
- Simple and user-friendly

## Technologies Used
- Python

## How to Run

1. Clone repository
2. Run:
   python chatbot.py

## Disclaimer
This is not a medical diagnosis tool. Always consult a doctor.
